# Product Requirements Document (PRD)

## Product

Nuvio

Version: 1.0 (MVP)

Status: Draft

---

# Objective

Build a platform that helps aspiring developers become job-ready through personalized guidance, structured learning paths, and measurable progress.

---

# Problem

Developers struggle because:

* Too many tutorials
* No clear roadmap
* No personalized guidance
* No way to measure progress
* Difficulty building job-ready skills

---

# Target Users

Primary users:

* Self-taught developers
* Diploma students
* Engineering students
* Career changers

---

# Goals

The MVP should allow a user to:

* Create an account
* Complete onboarding
* Take a skill assessment
* Receive a personalized roadmap
* Track learning progress

---

# Pages

* Landing Page
* Login
* Sign Up
* Onboarding
* Assessment
* Skill Report
* Dashboard
* Roadmap
* Profile
* Settings

---

# MVP Features

## Authentication

Users can:

* Sign up
* Log in
* Log out

---

## Onboarding

Collect:

* Career goal
* Experience level
* Daily study time

---

## Assessment

Evaluate:

* HTML
* CSS
* JavaScript
* Git
* React
* Node.js
* Express
* MongoDB

---

## Personalized Roadmap

Generate:

* Topics
* Projects
* Weekly goals

---

## Dashboard

Display:

* Progress
* Daily mission
* Current roadmap
* Job readiness score

---

# User Stories

As a new user,
I want to create an account,
so I can save my progress.

As a learner,
I want a personalized roadmap,
so I know what to study next.

As a student,
I want to track my progress,
so I can see how I'm improving.

---

# Success Criteria

The MVP is successful if users can:

* Register successfully
* Complete onboarding
* Finish an assessment
* Receive a roadmap
* Return and continue learning

---

# Out of Scope (Not in MVP)

* AI Mentor
* Resume Review
* Mock Interviews
* Community
* Mobile App
* Company Hiring Portal

---

# Release Goal

Launch a stable MVP that solves one problem exceptionally well:

Helping developers know exactly what to learn next.
